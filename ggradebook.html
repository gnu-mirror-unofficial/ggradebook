introduction to Ggradebook

This is a web page about Ggradebook. Ggradebook is the fully-featured GNU gradebook; an application for tracking student grades for teachers. It uses GTK+ and can optionally be compiled to use GNOME. The Ggradebook website can be found at http://Ggradebook.sourceforge.net/.
The author and maintainer of Ggradebook is Norbert de Jonge <norbertdejonge@yahoo.com>.

downloading Ggradebook

The main ftp distribution site of Ggradebook is ftp://ftp.gnu.org/gnu/ggradebook/. You may want to use one of the mirrors, though.
The website and main html distribution site of Ggradebook can be found at http://Ggradebook.sourceforge.net/.
